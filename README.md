# Приложение для изучения английского языка EnglishCard

Веб-приложение для изучения английских слов, разработанное в рамках курсовой работы. Позволяет пользователям изучать слова, добавлять новые, удалять текущие и отслеживать статистику.

## Требования к ПО
* **Язык программирования:** Python 3.12+
* **База данных:** PostgreSQL
* **Фреймворк:** Streamlit

## Настройка Базы Данных

1. В СУБД PostgreSQL (например, через PgAdmin) необходимо создать базу данных `english_card_db`.
2. Таблицы и базовый набор слов создаются автоматически при первом входе пользователя в приложение, либо путем выполнения SQL-запросов из модуля `db.py`.

### Схема базы данных
В проекте реализовано 3 таблицы со связью "многие ко многим" для изоляции пользовательских словарей:
* `users` — хранение пользователей.
* `words` — хранение общего словаря.
* `user_words` — таблица связи, хранящая статистику по каждому слову конкретного пользователя.

##  Установка и запуск проекта

1. Склонируйте репозиторий на локальный компьютер:
```bash
git clone [https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ.git](https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ.git)
```

2. Перейдите в папку проекта, создайте и активируйте виртуальное окружение:
```bash
python -m venv venv

# Команда для Windows:
venv\Scripts\activate

# Команда для Linux/Mac:
source venv/bin/activate
```

3. Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

4. Запустите веб-интерфейс приложения:
```bash
streamlit run main.py
```
